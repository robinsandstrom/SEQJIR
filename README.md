# SEQIJR model for Covid-19

SEQIJR-Model:

 S - Suspectible
 E - Exposed
 Q - Quarantine
 I - Infective
 J - Isolated
 R - Recovered

## About

This project was made to find a suitable model for estimating CoVid-19 outbreaks. Forks and improvements are welcomed and encouraged.
